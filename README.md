# Chat UI Simulation (JavaScript)

A modern and stylish chat interface built with pure JavaScript, HTML, and CSS.  
Simulates real-time messaging with typing effects, local chat history, and export options.

## Features
- Send and receive simulated messages
- Typing animation for bot replies
- Local conversation history (saved in browser)
- Multiple chats support
- Attach image preview
- Export chat as CSV
- Auto-scroll and Enter-to-send
